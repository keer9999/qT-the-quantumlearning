Single-Qubit Gates

⊗

Pauli-X (NOT Gate)

Flips the qubit state. Rotates π radians around X-axis on Bloch sphere.

Matrix:

[[0, 1], [1, 0]]

Maps |0⟩ → |1⟩ and |1⟩ → |0⟩

⊗

Pauli-Y Gate

Rotates π radians around Y-axis on Bloch sphere.

Matrix:

[[0, -i], [i, 0]]

Combines bit flip and phase flip

⊗

Pauli-Z Gate

Applies phase flip. Rotates π radians around Z-axis on Bloch sphere.

Matrix:

[[1, 0], [0, -1]]

Maps |0⟩ → |0⟩ and |1⟩ → -|1⟩

H

Hadamard Gate

Creates equal superposition. Fundamental gate for quantum computing.

Matrix:

1/√2 × [[1, 1], [1, -1]]

Maps |0⟩ → (|0⟩+|1⟩)/√2

S

S Gate (Phase Gate)

Applies π/2 phase shift. Quarter turn around Z-axis.

Matrix:

[[1, 0], [0, i]]

Rotates 90° around Z-axis

T

T Gate

Applies π/4 phase shift. Eighth turn around Z-axis.

Matrix:

[[1, 0], [0, e^(iπ/4)]]

Rotates 45° around Z-axis

Rx

RX Gate

Rotation around X-axis by angle θ.

Matrix:

[[cos(θ/2), -i·sin(θ/2)], [-i·sin(θ/2), cos(θ/2)]]

Parameterized rotation gate

Ry

RY Gate

Rotation around Y-axis by angle θ.

Matrix:

[[cos(θ/2), -sin(θ/2)], [sin(θ/2), cos(θ/2)]]

Parameterized rotation gate

Rz

RZ Gate

Rotation around Z-axis by angle θ.

Matrix:

[[e^(-iθ/2), 0], [0, e^(iθ/2)]]

Parameterized rotation gate

Two-Qubit Gates

⊕

CNOT (CX Gate)

Controlled-NOT gate. Flips target qubit if control qubit is |1⟩.

Action:

|a,b⟩ → |a, a⊕b⟩

Creates entanglement

⇄

SWAP Gate

Exchanges states of two qubits.

Action:

|a,b⟩ → |b,a⟩

Qubit exchange operation

Z

CZ Gate

Controlled-Z gate. Applies phase if both qubits are |1⟩.

Action:

Applies phase to |11⟩ state

Creates entanglement

iS

iSWAP Gate

Swaps qubits and applies phase.

Action:

Swap with imaginary phase

Superconducting qubit native gate

XX

XX Coupling Gate

Two-qubit interaction along X-axis.

Action:

Parameterized XX interaction

Native gate on some platforms

YY

YY Coupling Gate

Two-qubit interaction along Y-axis.

Action:

Parameterized YY interaction

Native gate on some platforms

ZZ

ZZ Coupling Gate

Two-qubit interaction along Z-axis.

Action:

Parameterized ZZ interaction

Native gate on some platforms

Three-Qubit Gates

T

Toffoli Gate (CCX)

Controlled-Controlled-NOT. Universal for classical computation.

Action:

Flips target if both controls are |1⟩

Quantum AND gate

F

Fredkin Gate (CSWAP)

Controlled-SWAP. Swaps two qubits if control is |1⟩.

Action:

Conditional qubit swap

Quantum multiplexer
